IF YOU GET STUCK
═══════════════════════════════════════════════════════════════════

Problem: "Supabase library not loaded"
→ Solution: Check Supabase CDN script is before supabase-client.js

Problem: "CORS Error"
→ Solution: Add your domain to Supabase Settings → API → CORS

Problem: No recipes showing
→ Solution: Verify API keys are correct, verify database has data

Problem: Forms don't submit
→ Solution: Add data-form-type="contact" or "suggestion" to forms

Problem: Can't find error
→ Solution: Check browser console (F12 → Console tab)

For more help:
→ See SUPABASE_QUICK_START.md Troubleshooting section
