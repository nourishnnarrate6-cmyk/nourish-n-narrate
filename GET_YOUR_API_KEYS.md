# 🔑 How to Get Your Supabase API Keys

## Step-by-Step Instructions

### Step 1: Go to Supabase
1. Open your web browser
2. Go to: **https://supabase.com**
3. Click **"Sign In"** (top right)

---

### Step 2: Sign In or Create Account
**If you already have an account:**
- Enter your email and password
- Click "Sign In"

**If you DON'T have an account:**
- Click "Sign Up"
- Enter your email
- Create a password
- Verify your email (check your inbox)

---

### Step 3: Create a New Project
1. Once logged in, you'll see your dashboard
2. Click **"New Project"** button
3. Fill in the form:
   - **Name:** `nourish-n-narrate` (or any name you like)
   - **Database Password:** Create a strong password (you won't need it again, but save it)
   - **Region:** Choose closest to you (or just pick US East if unsure)
4. Click **"Create new project"**
5. **Wait 1-2 minutes** for the project to be set up

---

### Step 4: Get Your API Keys
Once the project is created:

1. Look at the left sidebar
2. Click the **Settings icon** (gear icon at the bottom)
3. Click **"API"** in the menu
4. You'll see a section called **"Project API keys"**

---

### Step 5: Copy Your Keys
You need to copy TWO things:

**First, copy the URL:**
- Look for **"Project URL"** (under "Configuration" section)
- It looks like: `https://xxxxx.supabase.co`
- Click the copy button (small icon next to it)
- Paste it here: `SUPABASE_URL = `

**Second, copy the Key:**
- Look for **"anon public"** (under "Project API keys")
- Click the copy button next to it
- It looks like: `eyJhbGc...` (very long string)
- Paste it here: `SUPABASE_ANON_KEY = `
